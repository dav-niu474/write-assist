# 素材灵感助手 - Chrome 浏览器插件

一键收藏网页内容到素材灵感库。

## 功能特性

- 📄 **收藏页面**: 右键点击页面空白处，选择"收藏当前页面"
- 📝 **收藏选中文本**: 选中文字后右键选择"收藏选中文字"
- 🖼️ **收藏图片**: 在图片上右键选择"收藏图片"
- 🔗 **收藏链接**: 在链接上右键选择"收藏链接"
- 💬 **浮动按钮**: 页面右下角显示快速收藏按钮

## 安装方法

### 开发者模式安装

1. 打开 Chrome 浏览器，访问 `chrome://extensions/`
2. 打开右上角的"开发者模式"
3. 点击"加载已解压的扩展程序"
4. 选择 `chrome-extension` 文件夹
5. 安装完成！

### 配置

1. 点击浏览器工具栏的插件图标
2. 输入素材库 API 地址（默认: `http://localhost:3000`）
3. 点击"测试连接"确认服务正常
4. 保存设置

## 使用方法

### 方法一：右键菜单

在网页上右键，选择对应的收藏选项。

### 方法二：工具栏图标

点击浏览器工具栏的插件图标，使用快捷操作。

### 方法三：浮动按钮

点击页面右下角的绿色"+"按钮，快速收藏当前页面。

## 目录结构

```
chrome-extension/
├── manifest.json          # 插件配置文件
├── background/
│   └── background.js      # 后台脚本
├── content/
│   ├── content.js         # 内容脚本
│   └── content.css        # 内容样式
├── popup/
│   ├── popup.html         # 弹窗页面
│   ├── popup.js           # 弹窗脚本
│   ├── options.html       # 设置页面
│   └── options.js         # 设置脚本
└── icons/
    ├── icon16.svg
    ├── icon48.svg
    └── icon128.svg
```

## 注意事项

1. 确保素材库后端服务正在运行
2. 如果使用本地服务，需要在浏览器中允许访问 localhost
3. 部分网站可能限制内容脚本注入，导致插件无法使用

## 隐私说明

- 插件仅在用户主动操作时收集数据
- 所有数据发送到用户配置的 API 服务器
- 不收集任何用户浏览历史或个人信息
- 开源代码，可自行审查和修改
